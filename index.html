JReadline


JReadline is a Java library that provides an API which allow the programmer to write command-based programs in a terminal-independent manner. JReadline support most GNU Readline features.

Features:

  • Line editing
  • History (search, persistence)
  • Completion
  • Masking
  • Undo and Redo
  • Paste buffer
  • Emacs and Vi editing mode
  • Support POSIX OS's and Windows
  • Support out and error streams
  • Redirect
  • Pipeline

- and its very configurable
