h1. FAQ


h3. How is JReadline different from Jline1/2?

The JReadline started out of some of the shortcomings of Jline and its API is somewhat similar for getting output from console. Completion/Settings/etc is howevery different but changing from Jline to JReadline should not require much work.

h3. I found an issue, where can I report it?
